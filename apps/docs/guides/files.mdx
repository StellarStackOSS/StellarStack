# File Management

Access and manage your server files through the web interface or SFTP.

<Info>
  This guide is currently being expanded. File management features are available in the web panel under the Files tab.
</Info>

## Web File Manager

The web-based file manager allows you to:

- Browse server directory structure
- Upload and download files
- Create and delete files/folders
- Edit text files directly
- Manage permissions

## SFTP Access

For advanced users, SFTP access is available:

- **Host**: Your server address
- **Port**: 2022 (default)
- **Username**: Server username
- **Password**: SFTP password

## Common File Operations

- Configuration files (server.properties, config.yml, etc.)
- Plugin/mod management
- World/save data management
- Log file access
