# Server Management

Comprehensive guide for creating, configuring, and managing game servers.

<Info>
  This guide is currently being expanded. Please refer to the [Quickstart Guide](/guides/quickstart) for basic server creation.
</Info>

## Server Lifecycle

Servers have several states throughout their lifecycle:

- **Creating** - Server is being provisioned
- **Stopped** - Server is installed but not running
- **Running** - Server is active and accepting connections
- **Restarting** - Server is being restarted
- **Deleting** - Server is being removed

## Creating a Server

See the [Quickstart Guide](/guides/quickstart) for step-by-step instructions on deploying your first server.

## Managing Server Settings

You can adjust server settings including:

- Resource allocation (CPU, RAM, Disk)
- Startup command and environment variables
- Port mappings and network configuration
- Auto-restart settings

## Server Console

Access the live console to:

- View real-time server logs
- Execute server commands
- Monitor performance metrics
- Manage players and moderation
